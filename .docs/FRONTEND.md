### Frontend Robolife2

### Платформа

- React 18.2.0
- Redux 4.2.0
- Mui 5.8.6
- Amcharts 5.2.31
- RSuite 5.22.2
- Axios 1.1.3

### Настройка frontend-a

- Необходимо установить node.js v16.13.2.
- В директории frontend необходимо применить команду ```npm ci```.

### Запуск

```bash
npm run start
```
